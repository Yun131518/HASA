## ⚡ Quick Start

```bash
java -jar HASA_EXE.jar
```

```

║   HASA CLI - secp256k1 서명 도구     ║
╚══════════════════════════════════════╝

  1. 키페어 생성   (Generate Key Pair)
  2. 메시지 서명   (Sign Message)
  3. 서명 검증     (Verify)
  4. 파일 서명     (Sign File)
```
